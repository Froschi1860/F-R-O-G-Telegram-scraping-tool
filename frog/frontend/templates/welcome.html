Getting started

To get started sign up on Telegram with your phone number, get an API ID and register your credentials in the Credentials section. The username is necessarybut can be chosen freely and does not have an effect on the scraping process. After a set of credentials was saved it must be authorized by clicking the respective button. A one-time code is sent via the Telegram app which must be entered in F-R-O-G (Two-Factor-Authorization is not supported as of now). Once authorized successfully, F-R-O-G remembers this state and verifies it for each saved set of credentials on startup. Multiple phone numbers can be used for parallel scraping.
